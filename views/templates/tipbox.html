  • Type ctrl+1 to see some quick keyboard shortcuts.
  • You can easily submit a comment by hitting tab.
  • You can format your comments with Markdown. Read about Markdown here.
  • Click and drag to select lines to leave a comment.
  • In IGs and don't want to accidentally create a comment? Hit 'Read Only'
  • Want to see all the files in this submissions? Hit 'All files'
  • Want a quick preview of only your comments? Hit 'Hide code'
  • Want to color code your comments? Start a comment with !color with colors = [red, green, blue, yellow]. E.g. '!red Bug...'
